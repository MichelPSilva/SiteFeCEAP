Para funcionar nos pcs:

Apague a pasta node_modules
Apague os arquivos .json

inicie o projeto com o comando: npm init -y
instale o express com o comando: npm i express
instale o nodemon com o comando: npm i nodemon
instale o nunjucks com o comando: npm i nunjucks
modifique no arquivo package.json a linha com "node server.js" para "nodemon server.js"
execute o comando npm start
acesse no navegador pela url localhost:3000
